üêç Step 2: Python AI Service (Port 8000)
Open a terminal in the /python folder.

Install dependencies:

Bash
pip install -r requirements.txt
Run the server:

Bash
python -m uvicorn main:app --reload --port 8000
Important: Open a new terminal and expose this port via ngrok:

Bash
ngrok http 8000
Copy the generated https://... URL for the next step.

‚òï Step 3: Java Spring Boot Backend (Port 8080)
Open the project in VS Code (ensure the Extension Pack for Java is installed).

Go to src/main/java/com/splitbite/controller/OrderController.java.

Update the pythonServiceUrl with your Python ngrok URL from Step 2.

Run the application via the Run button or F5.

Important: Open a new terminal and expose this port via ngrok:

Bash
ngrok http 8080
Copy this https://... URL; this is your Main API URL.

üíª Step 4: React Frontend (Vite)
Open a terminal in the /frontend folder.

Install dependencies:

Bash
npm install
Update the NGROK_URL in join.jsx, create.jsx, and TrackingPage.jsx with your Java ngrok URL from Step 3 [cite: 2026-02-12, 2026-02-18].

Launch the frontend:

Bash
npm run dev
